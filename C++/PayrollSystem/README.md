# Payroll System
A system for handling payroll for enterprise accounts

## Structure
- Payroll System
  - Configuration
    - Pay Period Delimitation/Definition
  - User Accounts
    - Name
    - Tax Info
      - TPIN
      - Witholdings Info
    - Pay Rate
  - Clock Events
    - Associated User
    - Start
    - Stop
  - Salary
    - Working Hours
